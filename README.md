# Germán Valenzuela — Portfolio

Landing page personal para Germán Alfredo Valenzuela González, diseñador gráfico senior con 15+ años de experiencia.

## Estructura

```
german-portfolio/
└── index.html   ← Archivo único, contiene todo (HTML + CSS + JS + foto)
```

## Publicar en GitHub Pages

1. Crear repositorio en GitHub (ej: `german-portfolio`)
2. Subir `index.html` al repositorio
3. Ir a **Settings → Pages**
4. En *Branch*, seleccionar `main` y carpeta `/ (root)`
5. Guardar — en 1-2 minutos el sitio estará en:
   `https://tu-usuario.github.io/german-portfolio`

## Stack

- HTML / CSS / JS vanilla — sin dependencias
- Fuentes: Playfair Display · Space Grotesk · DM Mono (Google Fonts)
- Foto embebida en base64
